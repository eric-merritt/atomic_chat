class::frontend/src/atoms/message.ts::ToolCallPair | This interface defines the structure for a tool call pair, including the tool name, parameters, result, status, and content offset. It is used to track the state and position of a tool execution within a message.
class::frontend/src/atoms/message.ts::Message | This interface defines the structure for a message, including its ID, role, content, images, tool calls, tool pairs, and timestamp. It serves as the main data model for a chat message in the application.
function::frontend/src/atoms/message.ts::genId | This function generates a unique string ID for a message by combining the current timestamp with an incrementing counter. It ensures that each message has a distinct identifier.
function::frontend/src/atoms/message.ts::createMessage | This function creates a new Message object with the provided role and content, initializing other fields with default values. It uses genId to generate a unique ID and sets the timestamp to the current time.
function::frontend/src/atoms/message.ts::createMessageFromHistory | This function creates a Message object from a history entry, mapping the role and content fields. It initializes other fields with default values and sets the timestamp to 0.
