function::auth/credentials.py::remove_credential | This function removes a credential entry from the stored credentials dictionary using the provided alias. It saves the updated credentials to storage and returns True if the alias was found and deleted, otherwise it returns False.
function::auth/credentials.py::get_credential | This function retrieves a specific credential entry from the loaded credentials dictionary using the provided alias. It returns the full dictionary entry if found, or None if the alias does not exist.
function::auth/credentials.py::list_credentials | This function loads all stored credentials and creates a list of dictionaries with sensitive information masked. It returns the list of masked credential entries for safe display.
function::auth/credentials.py::build_auth_headers | This function retrieves a credential by alias and constructs an HTTP headers dictionary based on the credential type. It supports basic auth, bearer tokens, API keys, and cookies, raising an error if the credential is not found.
function::auth/credentials.py::_cli | This function sets up a command-line interface with subcommands for adding, listing, removing, and setting a master password for credentials. It parses arguments, handles user input for passwords, and executes the corresponding credential management operations.
