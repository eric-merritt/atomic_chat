function::context/tokens.py::_render_qwen_template | This function formats a list of message dictionaries into a Qwen3.5 chat template string. It handles role extraction, content normalization, and function call insertion before joining the parts.
function::context/tokens.py::_tokenize_via_server | This function attempts to count tokens by sending the text to a running llama-server via HTTP POST. It returns the token count if successful, or None if the server is unavailable or returns an error.
function::context/tokens.py::_get_model_path | This function retrieves the file path for the default model by checking environment variables and the MODELS configuration. It returns the path string if found, or None otherwise.
class::context/tokens.py::TokenCounter | This class provides a wrapper for token counting that prefers a remote server call but falls back to a local vocab-only model. It initializes the local model if available and exposes methods to count tokens for text or message lists.
function::context/tokens.py::__init__ | This method initializes the TokenCounter instance by setting the model path and resetting the model reference. It then calls the private initialization method to load the local model if the path exists.
