function::auth/conv_store.py::list_conversations | This function retrieves a paginated list of conversations for a specific user, supporting optional filtering by folder and search terms. It handles different storage backends and returns the conversation data along with pagination metadata.
function::auth/conv_store.py::get_conversation | This function fetches a single conversation by ID for a specific user, including its messages with pagination support. It returns the conversation details and message list if found, or None if the conversation does not exist.
function::auth/conv_store.py::update_conversation | This function updates the title or folder of an existing conversation for a specific user. It validates ownership, updates the fields, sets the current timestamp, and returns the updated conversation data.
function::auth/conv_store.py::delete_conversation | This function deletes a specific conversation by ID for a given user after verifying ownership. It returns True if the deletion was successful and False if the conversation was not found.
function::auth/conv_store.py::add_message | This function adds a new message to an existing conversation for a specific user. It validates the conversation exists, saves the message with optional images and tool calls, updates the conversation timestamp, and returns the new message data.
