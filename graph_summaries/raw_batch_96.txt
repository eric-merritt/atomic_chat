class::tools/accounting.py::RegisterInventoryItemTool | This class defines a tool for registering new inventory items with a unique SKU and description. It specifies the item type as either goods or service for subsequent accounting operations.
function::tools/accounting.py::_receive_inventory_impl | This function retrieves the user's ledger and validates the existence of the specified inventory item. It checks for errors related to missing ledgers or inactive items before proceeding.
class::tools/accounting.py::ReceiveInventoryTool | This class defines a tool to receive inventory and automatically record the corresponding purchase journal entry. It accepts parameters for the item SKU, quantity, and unit cost to process the transaction.
function::tools/accounting.py::_list_inventory_items_impl | This function retrieves the user's ledger and queries all associated inventory items. It calculates the total quantity on hand for each item by summing the remaining quantities from its layers.
class::tools/accounting.py::ListInventoryItemsTool | This class defines a tool that lists all inventory items along with their current quantity on hand. It executes the implementation function using the current user's database session.
function::tools/accounting.py::_deactivate_inventory_item_impl | This function retrieves the user's ledger and finds the specified inventory item by SKU. It checks if there is any remaining quantity in the item's cost layers before allowing deactivation.
class::tools/accounting.py::DeactivateInventoryItemTool | This class defines a tool to deactivate an inventory item by its SKU. It rejects the operation if the item still has any remaining quantity in its cost layers.
