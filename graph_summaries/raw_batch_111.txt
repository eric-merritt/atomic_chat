function::tools/mcp.py::call | This function parses JSON input to extract a URL, tool name, and parameters for an MCP tool call. It validates the input format and ensures the URL starts with http or https before proceeding.
class::tools/native.py::GetParamsTool | This class defines a tool that looks up and returns the parameter schema for a specified tool name. It retrieves the tool class from the registry and extracts its properties and required fields.
function::tools/native.py::call | This function retrieves the parameter schema for a given tool name from the registry. It constructs a list of parameter details including name, type, description, and whether it is required.
class::tools/native.py::ListToolsTool | This class defines a tool that lists available tools grouped by category, optionally filtering by a query string. It provides a description of its purpose and defines an optional query parameter.
function::tools/native.py::call | This function parses an optional query parameter to filter the list of available tools. It combines native tools with workflow groups and returns a filtered list based on the query.
function::tools/native.py::_with_descs | This helper function takes a list of tool names and returns a dictionary mapping each name to its description. It retrieves the description from the TOOL_REF dictionary for each provided name.
function::tools/native.py::_filter_catalog | This function filters the tool catalog to return the most relevant tools based on keywords from a description. It calculates a relevance score for each tool and returns the top matches.
function::tools/native.py::_route_tools | This function uses an LLM to determine which tools are needed for a specific task description. It sends the task and a filtered tool catalog to the server and parses the response.
class::tools/native.py::NeedToolTool | This class defines a tool that requests specific tool names based on a task description. It uses a router to identify the necessary tools and makes them available for subsequent calls.
function::tools/native.py::call | This function validates the input description and calls the router to get the required tool names. It returns an error if the description is missing or if the router fails to identify any tools.
