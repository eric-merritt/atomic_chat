class::auth/models.py::OAuthToken | This class defines a SQLAlchemy model for the oauth_tokens table with fields for user, provider, and tokens. It establishes a relationship with the User model to allow back-population of oauth tokens.
class::auth/rate_limit.py::_SlidingWindow | This class implements a thread-safe sliding window rate limiter using deques to store request timestamps. It provides a hit method to record requests and check if they are allowed within the specified window and max hits.
function::auth/rate_limit.py::__init__ | This method initializes the _SlidingWindow instance with the window size and maximum hit count. It also sets up an empty dictionary for buckets and a threading lock for thread safety.
function::auth/rate_limit.py::hit | This method records a request hit for a given key and checks if it exceeds the rate limit. It returns a tuple indicating if the request is allowed and the retry-after time if it is not.
function::auth/rate_limit.py::_client_ip | This function retrieves the client's IP address from the request headers. It attempts to parse the X-Forwarded-For header first, falling back to remote_addr or "unknown" if not available.
