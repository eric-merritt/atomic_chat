function::routes/conversations.py::get_conversation | This function retrieves a specific conversation by ID with pagination support from the store. It returns a 404 error if the conversation is not found, otherwise it returns the conversation data.
function::routes/conversations.py::update_conversation | This function updates an existing conversation with data provided in the request JSON. It returns a 404 error if the conversation is not found, otherwise it returns the updated conversation.
function::routes/conversations.py::delete_conversation | This function deletes a specific conversation by ID from the store. It returns a 404 error if the conversation is not found, otherwise it confirms the deletion.
function::routes/conversations.py::add_message | This function adds a new message to a specific conversation with details like role, content, and images. It returns a 404 error if the conversation is not found, otherwise it returns the created message with a 201 status.
function::routes/conversations.py::list_tasks | This function retrieves a list of tasks associated with a specific conversation. It returns a 404 error if the conversation is not found, otherwise it returns the list of tasks.
