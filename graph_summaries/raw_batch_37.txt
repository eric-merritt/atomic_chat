function::context/__init__.py::_db_row_to_qwen | This function converts a database row dictionary into a format compatible with the Qwen agent. It handles user, assistant, and tool roles, truncating tool results if necessary.
function::context/__init__.py::build_history | This function takes a list of database message dictionaries and converts them into Qwen-agent message dictionaries. It uses the _db_row_to_qwen helper function for each row in the list.
function::context/__init__.py::serialize_user_message | This function serializes a user message string into a dictionary format suitable for database storage. It sets the role to user and initializes an empty list for tool calls.
function::context/__init__.py::serialize_assistant_message | This function serializes an assistant message string and tool calls into a dictionary format for database storage. It ensures the tool calls list is not None by defaulting to an empty list.
function::context/__init__.py::serialize_tool_result | This function serializes a tool result into a dictionary format for database storage. It truncates the content if it exceeds the maximum character limit and includes the tool name and ID.
