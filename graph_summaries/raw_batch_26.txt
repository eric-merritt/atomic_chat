function::auth/rate_limit.py::rate_limit | This function acts as a decorator factory that creates a rate limiter for Flask views based on a sliding window algorithm. It restricts the number of requests per IP address within a specified time window and returns a 429 error if the limit is exceeded.
function::auth/rate_limit.py::decorator | This inner function wraps the target view function to apply the rate limiting logic. It returns the wrapper function that will handle the request interception and validation.
function::auth/rate_limit.py::wrapper | This function generates a unique key based on the endpoint and client IP to track request counts. It checks if the request is allowed, returns a 429 error with retry information if not, or executes the original view function if allowed.
function::auth/routes.py::_set_session_cookie | This function sets a session cookie on the HTTP response object with the provided session ID. It configures security flags such as httponly, samesite, and secure, along with the maximum age.
function::auth/routes.py::_hash_password | This function takes a plain text password string as input. It hashes the password using bcrypt with a generated salt and returns the resulting hash string.
