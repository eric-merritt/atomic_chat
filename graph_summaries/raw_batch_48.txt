function::frontend/src/api/preferences.ts::updateProfile | This function sends a PATCH request to the /api/auth/profile endpoint to update the user's username or email. It returns the parsed JSON response from the server.

function::frontend/src/api/system.ts::setSystemPrompt | This function sends a POST request to the /api/system endpoint to update the system prompt. It returns an object containing the updated prompt data or an error message if the request fails.
