class::frontend/src/atoms/conversation.ts::Conversation | This interface defines the structure for a conversation object, including metadata like id, title, and timestamps. It also specifies optional fields for folder and model information.
class::frontend/src/atoms/conversation.ts::ConversationMessage | This interface defines the structure for a single message within a conversation, including role, content, and timestamp. It also includes arrays for attached images and tool calls.
class::frontend/src/atoms/conversation.ts::ConversationListResponse | This interface defines the response structure for a list of conversations, including pagination details. It contains an array of Conversation objects along with total count, page, and limit.
class::frontend/src/atoms/message.ts::ImageAttachment | This interface defines the structure for an image attachment, including source URL, filename, and size in kilobytes. It is used to represent image data within messages.
class::frontend/src/atoms/message.ts::ToolCallInfo | This interface defines the structure for information about a tool call, including the tool name and input. It also includes optional fields for id and additional parameters.
