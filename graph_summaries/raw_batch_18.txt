function::auth/conversations_jsonl.py::get_conversation | This function retrieves a specific conversation's metadata and a paginated subset of its messages. It calculates the correct slice of the message list based on the provided page and limit parameters.
function::auth/conversations_jsonl.py::update_conversation | This function updates the title or folder of an existing conversation if those fields are provided in the input data. It also updates the timestamp and saves the modified metadata back to storage.
function::auth/conversations_jsonl.py::delete_conversation | This function deletes the metadata and message files associated with a specific conversation. It returns true if the files were successfully removed and false if the metadata file did not exist.
function::auth/conversations_jsonl.py::add_message | This function creates a new message object with a unique ID and timestamp, then appends it to the conversation's JSONL file. It also updates the conversation's metadata timestamp and saves it.
function::auth/conversations_jsonl.py::list_tasks | This function retrieves the list of tasks associated with a specific conversation from its metadata. It returns the tasks sorted by their creation timestamp in ascending order.
