function::auth/usage.py::estimate_tokens | This function provides a rough estimate of the number of tokens in a given text string. It calculates the length of the text divided by four, ensuring the result is at least one.
function::auth/usage.py::record_usage | This function records a usage event for a specific user, conversation, and model in the database. It handles the database session lifecycle by committing the transaction and closing the connection.
function::auth/usage.py::get_daily_usage | This function calculates the total number of tokens used by a user today in UTC time. It queries the database for usage events matching the user ID and current date, returning the sum of prompt and completion tokens.
function::auth/usage.py::quota_status | This function returns a dictionary containing the user's current usage statistics relative to a daily limit. It includes the used amount, limit, percentage, and boolean flags for warning and exhaustion states.
function::cli.py::_load_creds | This function attempts to load credentials from a JSON file specified by CREDS_PATH. It returns an empty dictionary if the file does not exist or if an error occurs during parsing.
